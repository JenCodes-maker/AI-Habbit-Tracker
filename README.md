# HabitAI - Smart Habit Tracking Web Application 🚀

HabitAI is a full-stack MERN application designed to help users build positive habits, track daily progress, and maintain consistency through interactive analytics and visual insights.

## ✨ Features

* 🔐 Secure User Authentication (JWT)
* 👤 Multi-User Support
* ➕ Create, Edit, and Delete Habits
* ✅ Mark Habits as Completed
* 🔥 Automatic Streak Tracking
* 📊 Weekly Progress Analytics with Charts
* 🟩 GitHub-Style Consistency Heatmap
* 📈 Completion Rate Tracking
* 🎯 Personalized Dashboard
* 💾 MongoDB Atlas Database Integration
* 🌐 Fully Deployed Web Application

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Chart.js
* React Calendar Heatmap
* React Router DOM

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

### Authentication

* JWT (JSON Web Tokens)
* bcryptjs

## 📸 Project Highlights

* Each user has their own private dashboard and habit data.
* Daily habit completion is visualized using a GitHub-style contribution heatmap.
* Interactive analytics help users monitor streaks and completion rates.
* Modern responsive UI designed for a smooth user experience.
